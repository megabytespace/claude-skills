# Frontend Design — The Anti-Slop Manifesto

## The Problem

Most AI-generated websites look the same: generic gradients, stock hero images, safe blue buttons, Lorem Ipsum vibes. This skill ensures every site is **distinctive, bold, and memorable**.

## Design Principles

### 1. Typography is 80% of Design
- **Body:** Sora, Inter, or Outfit — never system fonts alone
- **Headings:** Space Grotesk, Clash Display, or Cabinet Grotesk — geometric, bold
- **Mono:** JetBrains Mono — for code/data
- **Weights:** Full range (300-900). Light for body, Black for hero.
- **Size:** Hero h1: `clamp(2.5rem, 7vw, 5rem)` — big, confident, responsive
- **Spacing:** Negative on headings (-0.03em), positive on labels (0.15em)

### 2. Color is Emotion
- **Dark themes:** Near-black (#060610) with blue undertones, NOT pure #000
- **Accent:** ONE primary (cyan #00E5FF), ONE secondary (blue #50AAE3)
- **Glow:** `box-shadow: 0 0 24px rgba(accent, 0.15)` — subtle
- **Gradients:** Linear on CTAs, radial on backgrounds — 2 colors max
- **Text tiers:** Primary #f0f0f8, secondary #c0c0d8, muted #6e6e8e

### 3. Animation is Life
- **Scroll reveals:** opacity + translateY on IntersectionObserver
- **Hover:** EVERY interactive element — translate, scale, glow, color shift
- **Focus:** `outline: 2px solid accent; outline-offset: 3px` on focus-visible
- **Active:** `transform: scale(0.97)` on press
- **Page transitions:** fadeIn/fadeOut between pages
- **Hero:** Floating orbs OR particle canvas OR video background
- **Cards:** Glassmorphism glow via `::before`
- **Cursor:** `cursor: pointer` on EVERYTHING clickable

### 4. Real Content
- **Images:** Pexels API for stock, Ideogram/DALL-E for custom — never empty boxes
- **Video:** Pexels Videos API backgrounds — subtle, dark-filtered, looping
- **Copy:** Real words. Every word earns its place.

### 5. Layout
- Container: max-width 1140px, padding 0 2rem
- Alternate section backgrounds for visual rhythm
- Cards: `auto-fit, minmax(320px, 1fr)` responsive grid
- Full-bleed: maps, hero images, video backgrounds

### 6. Mobile First
- Test at 375px width
- Hamburger at 768px breakpoint
- Touch targets: min 44px
- Font scaling: clamp() on all headings

## Quick Checklist

- [ ] Hero has visual impact (video, particles, gradient, or image)
- [ ] 2-3 font families with clear hierarchy
- [ ] Consistent color palette (max 3 chromatic colors)
- [ ] Every link/button has hover + focus + active + cursor:pointer
- [ ] Scroll animations on section entry
- [ ] Page fadeIn on load
- [ ] Real images, compressed, lazy-loaded
- [ ] Mobile tested at 375px
- [ ] Footer complete (logo, nav, social, legal)
- [ ] No AI slop (generic gradients, stock-hero-with-overlay, safe-blue-button)

## The Apple Test

Before shipping: "Would Apple put this on their website?" If no, fix it.
